# Cheat Studio
A collection of video game cheats
<hr/>

## Components
### Autoclicker
- Save / load /delete click style
- Load default style
- CPS preview
- Obfuscating techniques
  - Timed stop (Programmable)
  - Variable frequency (Programmable)
> Keys (1 click on, 2 click off, change click auto change)
> - 'z': left click
> - 'x': right click
